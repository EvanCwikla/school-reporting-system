# SafeSchool AI 

An AI-powered anonymous reporting system designed to bridge the communication gap between students and school administration.

## Overview
SafeSchool provides a secure, encrypted platform for students to report safety concerns, bullying, or mental health issues. By leveraging **Gemini 3.1 Pro**, the system automatically triages incoming reports, summarizing content and flagging high-severity threats for immediate administrative action.

## Key Features
- **Anonymous Reporting:** Multi-step form with optional identity disclosure.
- **AI Triage:** Automated severity analysis (High/Medium/Low) using Google Gemini.
- **Admin Dashboard:** A "Priority Inbox" for staff to manage and resolve reports.
- **Modern UX:** Fully responsive UI built with Tailwind CSS and Framer Motion.

## Tech Stack
- **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **AI:** Google Generative AI (Gemini)
